OAuth

Plugin Identifier: 20000

Summary

This plugin is a combined edition of the cURL, JSON, ZIP, Common Crypto and MISC plugins. It includes all the functions necessary to implement OAuth 1.0 and 2.0.

Note: The plugin commands IDs have shift since the introduction of PKCS and RSA commands. If you have an old version of the plugin in your structure, remove it, reopen (to reset the tokens) install the new plugin and open again.

Change list for 2013-02-14

New Command

JSON Append array element

Creates a copy of an existing node and appends it as an array element to the specified node.

JSON Append array

Shorthand for Append node and SET TYPE to JSON_ARRAY.

Modified Command

libcurl version is updated to 7.29.0 better support for IMAP.

cURL now yields to 4D to avoid hang during long requests.

JSON CLOSE now checks if passed node is a valid root, to avoid over release.

JSON Append node now accepts a 3rd parameter, which can be a JSON text (object or array) to insert directly to an existing node.
